INFORMAES TCNICAS
Todo o cdigo foi formado acerca de 4 Activitys, todas interligadas
atravs da orientao a objetos, utilizando a linguagem Java para sua
construo. Tambm utilizamos de uma API, fazendo a conexo via banco de
dados com a aplicao, ou seja, para seu pleno funcionamento necessrio
que o smartphone esteja conectado a internet, caso contrrio no ser
possvel visualizar as listas de eventos disponibilizadas em nosso banco,
sendomontada atravs de funes assncronas para seu pleno funcionamento.
Para qualquer tipo de cadastro necessrio consultar um de nossos
administradores pelo suporte, visto que, por questes de segurana e
organizao, apenas pessoas autorizadas conseguem realizar a insero do
evento em nosso banco de dados atravs do "SuggarORM" plataforma
utilizada para armazenamento. No momento estamos trabalhando para que em
atualizaes futuras seja possvel que o usurio tenha controle de seus
prprios eventos, linkando o mesmo ao seu perfil, criando uma nova
comunidade, facilitando a busca por informaes em uma rede ampla e
acessvel a todos os interessados.
